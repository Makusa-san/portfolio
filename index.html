Interactive Elements

In-text link

Sample code:

<p>I want to see the <a href = "#" class="text-link">project roadmap</a> and explore other sections!</p>

Rendered element:

I want to see the project roadmap and explore other sections!

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">my projects</a>

Rendered element:

my projects

View Project Buttons

Sample code:

<button class = "project-preview-button" type = "preview">View Project</button>

Rendered element:

Image as a Link

Sample code:

<a href = "#" class = "image-link"><img src = "img/maple-cake.jpg" alt="a piece of maple syrup flavor cake lying on a plate"></a>

Rendered element:
